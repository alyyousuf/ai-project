# Sermon Translation Platform — Investor Demo (Saudi Arabia)

Prototype focused on Saudi Arabia. Built with Next.js + TypeScript + Tailwind.

Quick start:
1. cd sermon-translation-platform
2. npm install
3. npm run dev
4. Open http://localhost:3000

Primary colors:
- Primary: #175859
- Secondary: #C5984F

Notes:
- Default language: Arabic (RTL).
- Map center: Saudi Arabia (lat: 23.8859, lng: 45.0792).
- Use react-leaflet / Mapbox for interactive map (not included by default).
